Что выведет программа? Объяснить вывод программы.

```
package main

func main() {
    ch := make(chan int)
    go func() {
        for i := 0; i < 10; i++ {
            ch <- i
        }
    }()
 
    for n := range ch {
        println(n)
    }
}

```

Ответ:

```
При работе с каналами горутины блокируется на чтение или запись до тех пор пока они не будут прочитаны или записаны.
В данном примере будет deadlock, потому что основная горутина встанет бесконечное ожидание записи в канал

1. Для того что бы решить это проблему, нам нужно уведомить основную горутину, что больше не нужно ждать данные из канала.
Нужно закрыть канал после цикла, в котором записываются данные close(ch)
```
